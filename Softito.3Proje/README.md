## Tech Stack
*   **Backend:** C# .NET Core, Entity Framework Core (Code-First), ASP.NET Core Identity
*   **Database:** MSSQL Server
*   **Frontend (Admin):** HTML5, CSS3, Bootstrap, JavaScript, C3.js 
*   **Frontend (Public):** Custom Responsive Design
*   **Authentication:** ASP.NET Core Identity with EF Core store — cookie-based sign-in, password hashing, account lockout policy,
